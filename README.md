# fortigate-samba-ad-lab
## Enterprise Network Infrastructure Lab

## Overview

Project ini merupakan simulasi infrastruktur jaringan enterprise menggunakan FortiGate Firewall, Debian 13 Server, Samba Active Directory Domain Services (AD DS), dan Windows 10 Client.

Tujuan project ini adalah membangun layanan autentikasi terpusat menggunakan Active Directory dan mengintegrasikannya dengan infrastruktur jaringan berbasis FortiGate.


### Technologies

- FortiGate Firewall
- Debian 13
- Samba Active Directory
- DNS
- LDAP
- Kerberos
- Windows 10
- VMware Workstation
